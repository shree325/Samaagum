# Controllers Layer (HTTP Transport / API Routes)

This folder contains the HTTP controllers and routing configurations. Controllers handle requests from the client, call the appropriate services, and return HTTP responses.

## Deliverables for Interns:

1. **Controllers (`*.controller.ts`)**:
   - Extract parameters, query strings, and body data from Express requests.
   - Invoke service methods to process logic.
   - Respond with HTTP status codes and JSON data.
   - Example files:
     - `user.controller.ts`
     - `product.controller.ts`
     - `order.controller.ts`
