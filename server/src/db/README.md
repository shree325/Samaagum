# Database Layer (SQL Schemas & Migrations)

This folder is dedicated exclusively to database schemas and table DDL definitions.

## Deliverables for Interns:

1. **Schemas (`*.sql` or `*.schema.ts`)**:
   - Write your PostgreSQL table design DDL statements.
   - Example files:
     - `users.sql` (defining `users` table and indexes)
     - `products.sql` (defining `products` table and indexes)
     - `orders.sql` (defining `orders` and `order_items` tables, keys, and constraints)

*(Note: Write the TypeScript query logic in `src/repositories/` instead).*
